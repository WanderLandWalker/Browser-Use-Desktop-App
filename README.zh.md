# Browser Use Desktop App

> 在桌面上运行一组浏览器代理。

其他 AI 浏览器总想把浏览器和代理合二为一。你不用换 Chrome——这个只是代理端。

把你的 Cookie 导入全新的 Chromium，代理就能在你访问过的所有网站自动保持登录。还可以用全局快捷键从任何地方启动任务。

基于 [Browser Harness](https://github.com/browser-use/browser-harness) 构建。
